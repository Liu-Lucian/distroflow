# 🚀 一键完整设置 - 真正的 0 到 1

## 只需 1 个命令

```bash
./setup_and_launch.sh
```

## 这个脚本会自动完成

### 1. ✅ 收集 GitHub Token
- 提示你访问 https://github.com/settings/tokens
- 等待你输入 token

### 2. ✅ 创建 GitHub 仓库
- 自动检查仓库是否存在
- 如果不存在，通过 API 自动创建
- 设置仓库描述和基本信息

### 3. ✅ 配置 Git 认证
- 自动配置远程 URL（包含 token）
- 无需手动编辑任何文件

### 4. ✅ 推送第一个 Commit
- 检测本地未推送的 commits
- 自动推送到 GitHub
- 验证推送成功

### 5. ✅ 保存配置
- 将 token 保存到 `GITHUB_TOKEN.env`
- 下次可直接使用：`source GITHUB_TOKEN.env`

### 6. ✅ 启动发布系统
- 询问是否立即启动
- 可选择立即运行或稍后手动启动

## 如果遇到问题

### 问题 1: "仓库创建失败"
**原因**: Token 权限不足

**解决**:
1. 重新生成 token
2. 确保勾选 `repo` 权限（完整仓库访问）
3. 或者手动创建仓库：https://github.com/new

### 问题 2: "推送失败"
**原因**: Token 无效或网络问题

**解决**:
1. 检查 token 是否复制完整（通常以 `ghp_` 开头）
2. 确认 token 未过期
3. 检查网络连接

### 问题 3: "命令未找到"
**原因**: 脚本没有执行权限

**解决**:
```bash
chmod +x setup_and_launch.sh
./setup_and_launch.sh
```
